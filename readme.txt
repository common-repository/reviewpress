WordPress Reviews by ReviewPress is the best WordPress plugin for Implementing Reviews and Ratings in your WordPress projects.

== Description ==

WordPress Reviews by ReviewPress is the best WordPress plugin for Reviews and Ratings. It works for posts, pages and custom post types and includes Google reCaptcha as well. It provides you shortcodes and gives you complete control on output.


**Features:**

*  Super easy usage.
*  ShortCodes to display form and reviews.
*  Star and Heart Ratings.
*  Customizable and can be used on all post types.
*  HTML5 validation.
*  Custom CSS.


== Installation ==

This section describes how to install the plugin and get it working.

= 1) Install =

= Modern Way: =
1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "reviewpress".
3. Install, then Activate it.

= Old Way: =
1. Unzip (if it is zipped) and Upload `reviewpress` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= 2) Configure =
1. Reach out to the ReviewPress-> Settings Page. It contains all the settings. Checkout the tabs.
2. "Display Settings Tab", Change the Star or Heart, Change the color and allow authorization as well.
3. "Review Settings Tab", Change the order of reviews display.
4. "Review Form Settings Tab", Enable or disable the form elements and can apply captcha as well.
5. "Custom CSS Settings Tab", You can style it according to your needs.
